# Secure Notes App

A secure notes application built with **Next.js**, **Prisma**, **NextAuth**, and **PostgreSQL**, designed with security-first principles including authentication, authorization, and end-to-end encryption (E2EE).

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Prisma ORM**
- **PostgreSQL**
- **NextAuth.js**
- **End-to-End Encryption (E2EE)**

## Features

- Secure user authentication using NextAuth
- Role-based access control (RBAC) aligned with **IAM** and **least-privilege** principles
- Encrypted note handling to protect sensitive user data
- Protected API routes and server-side authorization checks
- Scalable and maintainable application architecture

## Security & Architecture

- Implemented authentication and authorization flows to restrict access based on user roles
- Analyzed application logic to identify and mitigate potential authorization gaps
- Applied least-privilege access controls across routes and APIs
- Documented system architecture, workflows, and data handling to support maintainability and onboarding
